Definition

A complex number $z$ is written as: $z = a + ib$, where:

$a$ → Real part of $z$ ($\mathcal{R}(z)$)
$b$ → Imaginary part of $z$ ($\mathcal{I}(z)$)

Example 1

Consider $z = 5 + 2i$:

$\mathcal{R}(z) = 5$
$\mathcal{I}(z) = 2$

Example 2

Consider $z = -3 + 7i$:

$\mathcal{R}(z) = -3$
$\mathcal{I}(z) = 7$

1. Cartesian / Rectangular Form

\(z = x + iy\)

\(x = \mathcal{R}(z)\) → Real part
\(y = \mathcal{I}(z)\) → Imaginary part

Example: \(z = 3 + 4i → x = 3, y = 4\)

Operations easy in Cartesian form: addition, subtraction, conjugate.

2. Polar Form

\(z = r(\cos\theta + i\sin\theta)\)

\(r = |z| = \sqrt{x^2 + y^2}\) → modulus
\(\theta = \arg(z) = \tan^{-1}(y/x)\) → argument

Use: Multiplication, division, powers, roots.

Example: \(1 + i = \sqrt{2}(\cos 45^\circ + i \sin 45^\circ)\)

3. Euler / Exponential Form

\(z = r e^{i\theta}\) (Euler’s formula: \(e^{i\theta} = \cos\theta + i\sin\theta\))

Multiplication: \(z_1 z_2 = r_1 r_2 e^{i(\theta_1 + \theta_2)}\)
Division: \(z_1 / z_2 = (r_1/r_2) e^{i(\theta_1 - \theta_2)}\)
Powers: \(z^n = r^n e^{in\theta}\)

4. Trigonometric / Sin-Cos Form

Essentially same as polar form: \(z = r(\cos\theta + i\sin\theta)\)

Often used in De Moivre’s Theorem.

5. Modulus-Argument Form

|z| = r, \arg(z) = \theta

Can write: \(z = |z|(\cos \arg(z) + i \sin \arg(z))\)

6. Conjugate Form

\(z = x + iy, \bar{z} = x - iy\)

Useful for division, real part extraction:

\(\mathcal{R}(z) = \frac{z + \bar{z}}{2}\)
\(\mathcal{I}(z) = \frac{z - \bar{z}}{2i}\)

7. Geometric / Vector Form

Consider \(z = x + iy\) as a vector in Argand plane: \(\vec{z} = \langle x, y \rangle\)

Modulus → magnitude of vector, Argument → angle with positive x-axis

JEE often uses this for loci, perpendicularity, collinearity.

8. Matrix Form (Advanced / Optional)

\(z = x + iy \equiv \begin{bmatrix} x & -y \\ y & x \end{bmatrix}\)

Multiplication of complex numbers ↔ matrix multiplication. Rare in JEE, mostly for Advanced derivations.

\(z_1 = x_1 + iy_1, z_2 = x_2 + iy_2\) are equal if \(x_1 = x_2 \text{ and } y_1 = y_2\)

Example: \(3 + 4i = 3 + 4i\) → True

\(z_1 + z_2 = (x_1 + x_2) + i(y_1 + y_2)\)

Example: \((2 + 3i) + (1 + 4i) = 3 + 7i\)

\(z_1 - z_2 = (x_1 - x_2) + i(y_1 - y_2)\)

Example: \((5 + 6i) - (2 + 3i) = 3 + 3i\)

\(z_1 z_2 = (x_1 x_2 - y_1 y_2) + i(x_1 y_2 + x_2 y_1)\)

Example: \((1 + 2i)(3 + 4i) = -5 + 10i\)

\(\frac{z_1}{z_2} = \frac{(x_1 x_2 + y_1 y_2) + i(y_1 x_2 - x_1 y_2)}{x_2^2 + y_2^2}, z_2 \neq 0\)

Example: \(\frac{1 + 2i}{1 + i} = 1.5 + 0.5i\)

\(\bar{z} = x - iy\)

Properties:

  • \(\overline{z_1 + z_2} = \bar{z_1} + \bar{z_2}\)
  • \(\overline{z_1 z_2} = \bar{z_1} \bar{z_2}\)
  • \(\overline{\bar{z}} = z\)

\(\arg(z) = \theta\) → angle with positive x-axis

Properties:

  • \(\arg(z_1 z_2) = \arg(z_1) + \arg(z_2)\)
  • \(\arg(z_1 / z_2) = \arg(z_1) - \arg(z_2)\)
  • \(\arg(\bar{z}) = -\arg(z)\)

Find the roots of the quadratic equation in complex numbers: \(z^2 - (3 + 4i)z + (5 + 6i) = 0\)

Solution:

Use quadratic formula: \(z = \frac{(3+4i) \pm \sqrt{(3+4i)^2 - 4(5+6i)}}{2}\)

Step 1: Compute discriminant: \((3+4i)^2 - 4(5+6i) = (9 + 24i - 16) - (20 + 24i) = -7 + 0i\)

Step 2: Square root of discriminant: \(\sqrt{-7} = i\sqrt{7}\)

Step 3: Apply formula: \(z = \frac{3+4i \pm i\sqrt{7}}{2}\)

Step 4: Final roots: \(z_1 = \frac{3 + 4i + i\sqrt{7}}{2} = \frac{3}{2} + i\frac{4+\sqrt{7}}{2}, \quad z_2 = \frac{3 + 4i - i\sqrt{7}}{2} = \frac{3}{2} + i\frac{4-\sqrt{7}}{2}\)

Find the modulus and argument of the complex number:

$z = -1 + \sqrt{3} i$

Step 1: Modulus:

$|z| = \sqrt{(-1)^2 + (\sqrt{3})^2} = 2$

Step 2: Argument:

$\theta = \tan^{-1}(y/x) = \tan^{-1}(\sqrt{3}/-1)$

Step 3: Quadrant check: x < 0, y > 0 → second quadrant:

$\theta = 180^\circ - 60^\circ = 120^\circ = \frac{2 \pi}{3}$

Final Answer:

$|z| = 2, \quad \arg(z) = \frac{2\pi}{3}$

Find the locus of the complex number $z = x + iy$ satisfying $|z - 1| = 2$.

Solution:

Step 1: Substitute $z = x + iy$: $|(x + iy) - 1| = 2$

Step 2: Simplify: $|(x - 1) + iy| = 2 \implies \sqrt{(x-1)^2 + y^2} = 2$

Step 3: Square both sides: $(x - 1)^2 + y^2 = 4$

Step 4: Interpretation: This represents a circle with center at $(1,0)$ and radius $2$ in the Argand plane.

Answer: Circle, center = (1,0), radius = 2

Solve the quadratic equation $z^2 - (3 - 4i)z + (5 - 12i) = 0$.

Solution:

Step 1: Use quadratic formula: $z = \frac{(3-4i) \pm \sqrt{(3-4i)^2 - 4(5-12i)}}{2}$

Step 2: Compute discriminant: $(3-4i)^2 - 4(5-12i) = (9 - 24i + 16i^2) - (20 - 48i) = (-7 - 24i) - (20 - 48i) = -27 + 24i$

Step 3: Compute square root of discriminant: Let $\sqrt{-27 + 24i} = a + bi$. Solve $a^2 - b^2 = -27, 2ab = 24 \implies a = 3, b = 4$ So, $\sqrt{-27 + 24i} = 3 + 4i$

Step 4: Roots: $z = \frac{(3-4i) \pm (3+4i)}{2}$

Step 5: Compute each root: $z_1 = \frac{(3-4i) + (3+4i)}{2} = \frac{6}{2} = 3$

$z_2 = \frac{(3-4i) - (3+4i)}{2} = \frac{-8i}{2} = -4i$

Answer: $z = 3, \; z = -4i$

If $z + \bar{z} = 6$ and $z\bar{z} = 10$, find $z$.

Solution:

Step 1: Let $z = x + iy$. Then $z + \bar{z} = 2x = 6 \implies x = 3$.

Step 2: $z \bar{z} = x^2 + y^2 = 10 \implies 3^2 + y^2 = 10 \implies y^2 = 1 \implies y = \pm 1$.

Answer: $z = 3 + i, \; 3 - i$

If $z_1 = 2(\cos 30^\circ + i \sin 30^\circ)$ and $z_2 = 3(\cos 45^\circ + i \sin 45^\circ)$, find $z_1 z_2$ in polar form.

Solution:

Step 1: Modulus of product: $|z_1 z_2| = |z_1||z_2| = 2 \cdot 3 = 6$.

Step 2: Argument of product: $\arg(z_1 z_2) = 30^\circ + 45^\circ = 75^\circ$.

Answer: $z_1 z_2 = 6(\cos 75^\circ + i \sin 75^\circ)$

Points $z_1 = 0$, $z_2 = 2$, and $z_3$ form an equilateral triangle. Find $z_3$.

Solution:

Step 1: Formula for rotation: $z_3 = z_1 + (z_2 - z_1)e^{i 60^\circ} = 0 + 2 e^{i\pi/3} = 2(\cos 60^\circ + i \sin 60^\circ)$

Step 2: Convert to rectangular form: $z_3 = 2(1/2 + i\sqrt{3}/2) = 1 + i\sqrt{3}$

Answer: $z_3 = 1 + i\sqrt{3}$

Solve for complex number $z$ given $|z - 1| = |z + 1|$.

Solution:

Step 1: Let $z = x + iy$, then equation becomes: $| (x+iy) - 1 | = | (x+iy) + 1 |$

Step 2: Simplify modulus: $\sqrt{(x-1)^2 + y^2} = \sqrt{(x+1)^2 + y^2}$

Step 3: Square both sides: $(x-1)^2 + y^2 = (x+1)^2 + y^2$

Step 4: Simplify equation: $x^2 - 2x + 1 + y^2 = x^2 + 2x + 1 + y^2 \implies -2x = 2x \implies x = 0$

Step 5: Conclusion: $x = 0 \implies z = iy, \text{ any real } y$

Answer: $z = iy, \; y \in \mathbb{R}$

Solve for $z$ if $z + \bar{z} = 4$ and $z \bar{z} = 5$.

Solution:

Step 1: Let $z = x + iy \implies \bar{z} = x - iy$.

Step 2: First equation $z + \bar{z} = 4 \implies (x + iy) + (x - iy) = 4 \implies 2x = 4 \implies x = 2$.

Step 3: Second equation $z \bar{z} = 5 \implies (x+iy)(x-iy) = x^2 + y^2 = 5$.

Step 4: Substitute $x = 2$: $2^2 + y^2 = 5 \implies 4 + y^2 = 5 \implies y^2 = 1 \implies y = \pm 1$.

Step 5: Write all solutions: $z = 2 + i$ or $z = 2 - i$.

Answer: $z = 2 + i, \; 2 - i$
