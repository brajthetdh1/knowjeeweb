What is Friction?

Friction is the force that opposes relative motion between two surfaces in contact.

Friction Force

f ≤ μN

μ = coefficient of friction

Key Characteristics:

• Parallel to contact surface

• Opposes relative motion

• Depends on nature of surfaces

• Independent of contact area

Friction effects:

Walking

Braking

Gripping

Types of Friction

Static Friction

✓ 0 ≤ fs ≤ μsN

✓ Opposes impending motion

✓ Self-adjusting force

✓ μs is maximum

Kinetic Friction

✓ fk = μkN

✓ Opposes actual motion

✓ Constant magnitude

✓ μk < μs

Rolling Friction

✓ fr = μrN

✓ Opposes rolling motion

✓ Smallest of all

✓ μr << μk

JEE Memory Tip: μstatic > μkinetic > μrolling - Remember this order!

Static Friction

Static Friction Range

0 ≤ fs ≤ μsN

Self-adjusting force up to maximum value

Key Features

✓ Adjusts to prevent motion

✓ Maximum at impending motion

✓ fs max = μsN

✓ Angle of friction: tanθ = μs

Examples

✓ Pushing heavy furniture

✓ Car on inclined road

✓ Ladder against wall

✓ Banking of roads

Kinetic Friction

Kinetic Friction Formula

fk = μkN

Constant magnitude, independent of speed

Characteristics

✓ Constant for given surfaces

✓ Independent of velocity

✓ Always opposes motion

✓ Causes energy dissipation

Applications

✓ Sliding objects

✓ Braking systems

✓ Sandpaper smoothing

✓ Matchstick ignition

Angle of Friction & Repose

Angle of Friction (λ)

✓ tanλ = μs

✓ Angle between resultant and normal

✓ R = √(N² + f²)

✓ Important for inclined planes

Angle of Repose (θ)

✓ tanθ = μs

✓ Maximum angle before sliding

✓ θ = λ (angle of friction)

✓ Used in material handling

JEE Strategy: For inclined plane problems, use μ = tanθ where θ is angle of repose

Friction on Inclined Plane

Critical Angle Formulas

θmax = tan⁻¹(μs)

a = g(sinθ - μkcosθ)

For motion down the incline

Up the Incline

F > mg(sinθ + μkcosθ)

a = [F - mg(sinθ + μkcosθ)]/m

Friction opposes motion

Down the Incline

mg sinθ > μkmg cosθ

a = g(sinθ - μkcosθ)

Friction opposes motion

JEE Example Problem 1

JEE Main 2023 Question:

A block of mass 2 kg rests on a rough surface (μs = 0.6, μk = 0.4). Find the minimum force required to move the block and the acceleration if force is 15 N.

Solution:

Step 1: Minimum force to move

fmax = μsN = 0.6 × 2 × 9.8 = 11.76 N

Fmin = 11.76 N

Step 2: For F = 15 N (kinetic friction)

fk = μkN = 0.4 × 19.6 = 7.84 N

Step 3: Net force and acceleration

Fnet = 15 - 7.84 = 7.16 N

a = Fnet/m = 7.16/2 = 3.58 m/s²

JEE Example Problem 2

JEE Advanced 2021 Question:

A block is placed on an inclined plane with μ = 1/√3. Find the angle at which the block just starts sliding and the acceleration if angle is 45°.

Solution:

Step 1: Angle of repose

θ = tan⁻¹(μ) = tan⁻¹(1/√3) = 30°

Block starts sliding at 30°

Step 2: At θ = 45°

a = g(sinθ - μcosθ)

a = 9.8(sin45° - (1/√3)cos45°)

Step 3: Calculate acceleration

a = 9.8(0.707 - 0.577 × 0.707)

a = 9.8 × 0.707 × (1 - 0.577)

a = 9.8 × 0.707 × 0.423 = 2.93 m/s²

Methods to Reduce Friction

Lubrication

✓ Oil, grease between surfaces

✓ Reduces μ significantly

✓ Prevents wear and tear

✓ Used in engines, machines

Ball Bearings

✓ Convert sliding to rolling

✓ μrolling << μsliding

✓ Used in wheels, motors

✓ Increases efficiency

JEE Note: Streamlining reduces fluid friction, polishing reduces surface roughness, and using wheels converts sliding to rolling friction.
