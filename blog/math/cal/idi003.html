Trick 1: Algebraic Division for Rational Functions

Key Concept:

When degree of numerator ‚â• degree of denominator, perform polynomial division first.

$$ \int \frac{P(x)}{Q(x)} dx = \int S(x) dx + \int \frac{R(x)}{Q(x)} dx $$

Where $P(x) = Q(x) \cdot S(x) + R(x)$ with deg(R) < deg(Q)

üìù Example:

Evaluate $\int \frac{x^3 + 2x + 1}{x^2 + 1} dx$

Step 1: Divide: $\frac{x^3 + 2x + 1}{x^2 + 1} = x + \frac{x + 1}{x^2 + 1}$

Step 2: Split integral: $\int x dx + \int \frac{x}{x^2 + 1} dx + \int \frac{1}{x^2 + 1} dx$

Step 3: Solve: $\frac{x^2}{2} + \frac{1}{2}\ln|x^2 + 1| + \tan^{-1}x + C$

üí° When to Use:

When numerator degree ‚â• denominator degree in rational functions

Trick 2: Trigonometric Substitution Mastery

Key Substitutions:

$\sqrt{a^2 - x^2}$
$x = a\sin\theta$
$dx = a\cos\theta d\theta$
$\sqrt{a^2 + x^2}$
$x = a\tan\theta$
$dx = a\sec^2\theta d\theta$
$\sqrt{x^2 - a^2}$
$x = a\sec\theta$
$dx = a\sec\theta\tan\theta d\theta$

üìù Example:

Evaluate $\int \frac{dx}{\sqrt{4 - x^2}}$

Step 1: Let $x = 2\sin\theta \Rightarrow dx = 2\cos\theta d\theta$

Step 2: $\sqrt{4 - x^2} = \sqrt{4 - 4\sin^2\theta} = 2\cos\theta$

Step 3: $\int \frac{2\cos\theta d\theta}{2\cos\theta} = \int d\theta = \theta + C$

Step 4: $\theta = \sin^{-1}(\frac{x}{2}) \Rightarrow \sin^{-1}(\frac{x}{2}) + C$

üí° Pro Tip:

Memorize the three standard forms and their substitutions

Trick 3: Partial Fractions - The Smart Way

Standard Forms:

‚Ä¢ $\frac{px + q}{(x - a)(x - b)} = \frac{A}{x - a} + \frac{B}{x - b}$

‚Ä¢ $\frac{px + q}{(x - a)^2} = \frac{A}{x - a} + \frac{B}{(x - a)^2}$

‚Ä¢ $\frac{px^2 + qx + r}{(x - a)(x^2 + bx + c)} = \frac{A}{x - a} + \frac{Bx + C}{x^2 + bx + c}$

üìù Example:

Evaluate $\int \frac{2x + 3}{(x - 1)(x + 2)} dx$

Step 1: $\frac{2x + 3}{(x - 1)(x + 2)} = \frac{A}{x - 1} + \frac{B}{x + 2}$

Step 2: $2x + 3 = A(x + 2) + B(x - 1)$

Step 3: Solve: $A = \frac{5}{3}, B = \frac{1}{3}$

Step 4: $\int \frac{5}{3(x - 1)} + \frac{1}{3(x + 2)} dx = \frac{5}{3}\ln|x - 1| + \frac{1}{3}\ln|x + 2| + C$

üí° Shortcut:

Use cover-up rule for linear factors: $A = \left.\frac{2x+3}{x+2}\right|_{x=1}$

üöÄ Essential Integration Formulas

Basic Integrals:

‚Ä¢ $\int x^n dx = \frac{x^{n+1}}{n+1} + C$ $(n \neq -1)$

‚Ä¢ $\int \frac{1}{x} dx = \ln|x| + C$

‚Ä¢ $\int e^x dx = e^x + C$

‚Ä¢ $\int a^x dx = \frac{a^x}{\ln a} + C$

‚Ä¢ $\int \sin x dx = -\cos x + C$

‚Ä¢ $\int \cos x dx = \sin x + C$

Advanced Formulas:

‚Ä¢ $\int \sec^2 x dx = \tan x + C$

‚Ä¢ $\int \csc^2 x dx = -\cot x + C$

‚Ä¢ $\int \sec x \tan x dx = \sec x + C$

‚Ä¢ $\int \frac{1}{\sqrt{1-x^2}} dx = \sin^{-1}x + C$

‚Ä¢ $\int \frac{1}{1+x^2} dx = \tan^{-1}x + C$
