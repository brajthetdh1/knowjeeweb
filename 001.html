ADVANCED ROTATIONAL DYNAMICS

COMPOUND PHYSICAL PENDULUM

A uniform rod of length L and mass M is pivoted at one end. A point mass m is attached at a distance x from the pivot. The system performs small oscillations. Find the time period of oscillation and determine the value of x for which the time period is minimum.

System Parameters

• Rod length: L, mass: M

• Point mass: m

• Distance from pivot: x

• Small oscillations

• Uniform gravitational field

Objectives

• Time period T as function of x

• Find x for minimum T

• Physical interpretation

• Special cases analysis

Physical Pendulum Fundamentals

What is a Physical Pendulum?

A physical pendulum is any rigid body suspended from a fixed point that oscillates under gravity.

Simple Pendulum

• Point mass on string

• T = 2π√(L/g)

• Idealized model

Physical Pendulum

• Real rigid body

• T = 2π√(I/mgd)

• Depends on mass distribution

General Formula

T = 2π√(I/mgd)

I

Moment of inertia about pivot

m

Total mass

d

Distance from pivot to CM

Step 1: Moment of Inertia About Pivot

Rod's Moment of Inertia

For uniform rod of mass M, length L, about one end:

I_rod = ML²/3

This is a standard result using integration or parallel axis theorem.

Point Mass Moment of Inertia

For point mass m at distance x from pivot:

I_mass = mx²

Total Moment of Inertia

Since both rotate about the same pivot:

I_total = I_rod + I_mass

I = ML²/3 + mx²

[Equation 1]

Step 2: Center of Mass Calculation

Individual Mass Positions

Rod's CM

Mass: M

Distance from pivot: L/2

Point Mass

Mass: m

Distance from pivot: x

Composite Center of Mass

Using weighted average:

d = (M × (L/2) + m × x) / (M + m)

Where d is distance from pivot to total center of mass.

[Equation 2]

Physical Significance

The center of mass position determines the restoring torque when the system is displaced.

Smaller d means larger restoring torque for same angular displacement.

Step 3: Time Period Expression

Physical Pendulum Formula

General time period for physical pendulum:

T = 2π√[I/(m_total × g × d)]

Substitute Our Values

Total mass: m_total = M + m

Moment of inertia: I = ML²/3 + mx²

CM distance: d = (ML/2 + mx)/(M + m)

T = 2π√[(ML²/3 + mx²) / ((M + m) × g × (ML/2 + mx)/(M + m))]

Simplified Expression

Notice (M + m) cancels in denominator:

T(x) = 2π√[(ML²/3 + mx²) / (g(ML/2 + mx))]

[Final Time Period Formula]

Step 4: Finding Minimum Time Period

Optimization Approach

We need to find x that minimizes T(x). Since T ∝ √f(x), minimizing T is equivalent to minimizing f(x):

f(x) = (ML²/3 + mx²) / (ML/2 + mx)

We'll use calculus: find df/dx = 0

Quotient Rule Application

For f(x) = u/v, where:

u = ML²/3 + mx²

v = ML/2 + mx

u' = 2mx

v' = m

Derivative Setup

Using quotient rule: f'(x) = (u'v - uv')/v²

We'll solve f'(x) = 0 in the next step.

Step 5: Solving f'(x) = 0

Setting Up the Equation

From f'(x) = (u'v - uv')/v² = 0, we get:

u'v = uv'

Substituting:

(2mx)(ML/2 + mx) = (ML²/3 + mx²)(m)

Simplifying the Equation

Divide both sides by m (m ≠ 0):

2x(ML/2 + mx) = ML²/3 + mx²

MLx + 2mx² = ML²/3 + mx²

MLx + mx² = ML²/3

Final Quadratic Equation

mx² + MLx - ML²/3 = 0

This is our key equation to solve for x.

Step 6: Solving the Quadratic

Quadratic Formula

For equation: mx² + MLx - ML²/3 = 0

Using quadratic formula: x = [-b ± √(b² - 4ac)] / 2a

a = m

b = ML

c = -ML²/3

Discriminant Calculation

Δ = b² - 4ac = (ML)² - 4(m)(-ML²/3)

Δ = M²L² + (4/3)mML²

= ML²[M + (4/3)m]

Roots of the Equation

x = [-ML ± √(ML²[M + (4/3)m])] / (2m)

x = [-ML ± ML√(1 + (4/3)(m/M))] / (2m)

Step 7: Physical Interpretation

Choosing the Physical Root

We have two roots:

x₁

Positive root

Physically meaningful

x₂

Negative root

Not physically possible

So we take the positive root:

x = [√(1 + (4/3)(m/M)) - 1] × (ML)/(2m)

Special Case: m = M

If point mass equals rod mass:

x = [√(1 + 4/3) - 1] × (ML)/(2M)

= [√(7/3) - 1] × L/2

≈ 0.527 × L/2 ≈ 0.264L

Physical Insight

The optimal position is closer to the pivot than the rod's center of mass.

This minimizes the moment of inertia to restoring torque ratio.

Step 8: Equivalent Simple Pendulum

Length of Equivalent Simple Pendulum

For any physical pendulum, we can find an equivalent simple pendulum with same period:

L_eq = I/(m_total × d)

Substituting our values:

L_eq = (ML²/3 + mx²) / (ML/2 + mx)

At Optimal x

When x is chosen for minimum period, L_eq is also minimized.

This gives the most "efficient" pendulum configuration.

Physical Significance

The equivalent length represents how "compact" the mass distribution is relative to the pivot.

Smaller L_eq means faster oscillations for the same restoring torque.

Final Results

COMPLETE SOLUTION

Time Period

T(x) = 2π√[(ML²/3 + mx²) / (g(ML/2 + mx))]

Optimal Position

x_opt = [√(1 + (4/3)(m/M)) - 1] × (ML)/(2m)

Special Cases

m → 0

x_opt → L/√3 ≈ 0.577L

(Rod alone case)

m → ∞

x_opt → L/2

(Point mass dominates)
