üéØ Core Concept: The Fundamental Formula

In an m √ó n grid, the number of shortest paths from bottom-left to top-right is given by:

$$ \text{Number of paths} = \binom{m+n}{m} = \binom{m+n}{n} = \frac{(m+n)!}{m! \cdot n!} $$

Why this works: Each shortest path consists of exactly m right moves and n up moves.

The total number of arrangements of these moves is the binomial coefficient.

Problem 1: Basic Grid Path

Find the number of shortest paths from point A(0,0) to point B(3,4) moving only right (R) and up (U).

Solution Approach:

Step 1: Identify moves needed: 3 Right (R) + 4 Up (U)

Step 2: Total moves = 3 + 4 = 7

Step 3: Apply formula: $\binom{7}{3} = \binom{7}{4}$

Step 4: Calculate: $\binom{7}{3} = \frac{7!}{3! \cdot 4!} = \frac{5040}{6 \cdot 24} = \frac{5040}{144} = 35$

Step 5: Answer: 35 paths

JEE Main 2021 Medium

Problem 2: Paths with Restrictions

Find the number of shortest paths from A(0,0) to B(5,5) that do NOT pass through point C(2,2).

Solution Approach:

Step 1: Total paths without restrictions: $\binom{10}{5} = 252$

Step 2: Paths passing through C(2,2):

‚Ä¢ A to C: $\binom{4}{2} = 6$ paths

‚Ä¢ C to B: $\binom{6}{3} = 20$ paths

‚Ä¢ Total through C: $6 \times 20 = 120$ paths

Step 3: Valid paths = Total - Through C = $252 - 120 = 132$

Step 4: Answer: 132 paths

JEE Advanced 2020 Hard

Problem 3: Paths Avoiding a Diagonal

Find the number of shortest paths from A(0,0) to B(6,6) that never go above the line y=x (diagonal).

Solution Approach:

Step 1: This is a classic Catalan number problem

Step 2: Total paths without restriction: $\binom{12}{6} = 924$

Step 3: Use reflection principle or Catalan formula:

‚Ä¢ Valid paths = $\frac{1}{6+1} \times \binom{12}{6} = \frac{1}{7} \times 924 = 132$

Step 4: Alternative: $C_6 = \frac{1}{7} \binom{12}{6} = 132$

Step 5: Answer: 132 paths

üöÄ Quick Solving Strategies

Key Formulas:

  • Basic grid: $\binom{m+n}{m}$
  • Avoiding a point: Total - Through point
  • Catalan paths: $\frac{1}{n+1}\binom{2n}{n}$
  • With obstacles: Inclusion-Exclusion

Problem-Solving Tips:

  • Always identify required R and U moves first
  • For restrictions, use complementary counting
  • Remember symmetry in binomial coefficients
  • Practice visualization of paths
