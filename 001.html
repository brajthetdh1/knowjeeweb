Distance vs Displacement

Distance is the total path length traveled by an object.

Displacement is the shortest straight-line path between initial and final positions.

Key Difference:

Distance is scalar (magnitude only), Displacement is vector (magnitude + direction).

JEE Significance: Fundamental kinematics concept tested every year.

Key aspects:

Path Length

Shortest Path

Calculation

Distance - Scalar Quantity

Distance measures the total path covered regardless of direction.

Properties

✓ Always positive

✓ Path-dependent

✓ SI unit: meter (m)

✓ Increases with motion

Examples

✓ Car odometer reading

✓ Marathon race route

✓ Hiking trail length

✓ GPS tracked path

JEE Memory Tip: Distance ≥ |Displacement| always. They're equal only for straight-line motion.

Displacement - Vector Quantity

Displacement measures change in position with direction.

Properties

✓ Can be positive/negative

✓ Path-independent

✓ SI unit: meter (m)

✓ Can be zero

Mathematical Form

Δr = r_final - r_initial

✓ Vector subtraction

✓ Direction matters

✓ Coordinate dependent

Distance vs Displacement: Key Differences

Distance (Scalar)

✓ Total path length

✓ Always positive

✓ Path dependent

✓ Never decreases

Displacement (Vector)

✓ Shortest path

✓ Can be zero/negative

✓ Path independent

✓ Can decrease

Important Relationship

Distance ≥ |Displacement|

Equality holds only for straight-line motion without direction change

JEE Example Problem 1

JEE Main 2022 Question:

A particle moves from point A to B (4m East), then to C (3m North). Calculate total distance and displacement.

Solution:

Step 1: Total Distance

Distance AB = 4m

Distance BC = 3m

Total Distance = 4 + 3 = 7m

Step 2: Displacement

Using Pythagoras theorem:

|Displacement| = √(4² + 3²) = √25 = 5m

Direction: tan⁻¹(3/4) = 36.87° North of East

JEE Example Problem 2

JEE Advanced 2020 Question:

A particle moves in a circle of radius R. Find distance and displacement for: (a) Half revolution (b) Full revolution

Solution:

(a) Half Revolution

Distance = Half circumference = πR

Displacement = Diameter = 2R

(b) Full Revolution

Distance = Circumference = 2πR

Displacement = 0 (returns to start)

Key Insight

Displacement can be zero even when distance is large!

Calculus Connection

From Velocity

Δr = ∫v dt

Displacement = Integral of velocity

Distance = ∫|v| dt

From Acceleration

v = ∫a dt

Then find displacement

JEE Advanced favorite

Key Insight: For variable acceleration, calculus methods are essential for finding displacement and distance.

Real World Applications

Navigation Systems

GPS shows displacement to destination, odometer shows distance traveled

Sports Analytics

Football: Total distance run vs net displacement from start position

Orbital Mechanics

Satellites: Large orbital distance, but displacement relative to Earth may be small

Weather Patterns

Storm tracking: Path length vs straight-line displacement

Common Misconceptions

Misconception

"Distance and displacement are always equal"

Truth: Equal only for straight-line motion without direction change

Misconception

"Displacement can never be greater than distance"

Truth: |Displacement| ≤ Distance always

Misconception

"Zero displacement means no motion"

Truth: Can have motion and return to start (circular path)

Misconception

"Distance decreases when returning"

Truth: Distance never decreases, displacement can
