Type 1: Common Tangents to Two Circles

Find all common tangents to the circles $x^2 + y^2 = 4$ and $x^2 + y^2 - 6x - 8y + 24 = 0$

Solution Approach:

Step 1: Standardize the second circle

$x^2 + y^2 - 6x - 8y + 24 = 0$

Complete the square: $(x^2 - 6x + 9) + (y^2 - 8y + 16) + 24 - 9 - 16 = 0$

$(x-3)^2 + (y-4)^2 = 1$

So we have circles: $C_1$: center $(0,0)$, radius $2$ and $C_2$: center $(3,4)$, radius $1$

Step 2: Distance between centers

$d = \sqrt{(3-0)^2 + (4-0)^2} = \sqrt{9 + 16} = 5$

Since $d = r_1 + r_2 = 2 + 1 = 3$? Wait, $5 > 3$, so circles are separate.

Number of common tangents = 4 (2 direct, 2 transverse)

Step 3: Find transverse common tangents

Let tangent be $y = mx + c$

For $C_1$: distance from center = $\frac{|c|}{\sqrt{1+m^2}} = 2$

For $C_2$: distance from center = $\frac{|3m + 4 - c|}{\sqrt{1+m^2}} = 1$

Solving these gives the transverse tangents.

Step 4: Direct common tangents

Use external center of similitude or solve similarly with same slope but different intercepts.

ðŸ’¡ Key Formula: Common Tangents to Two Circles

For circles with centers $C_1$, $C_2$ and radii $r_1$, $r_2$:

  • If $d > r_1 + r_2$: 4 common tangents
  • If $d = r_1 + r_2$: 3 common tangents
  • If $|r_1 - r_2| < d < r_1 + r_2$: 2 common tangents
  • If $d = |r_1 - r_2|$: 1 common tangent
  • If $d < |r_1 - r_2|$: 0 common tangents

Type 2: Common Tangents to Circle and Parabola

Find the common tangents to the parabola $y^2 = 4x$ and the circle $x^2 + y^2 - 2x - 4y + 4 = 0$

Solution Approach:

Step 1: Standard form of circle

$x^2 + y^2 - 2x - 4y + 4 = 0$

Complete squares: $(x^2 - 2x + 1) + (y^2 - 4y + 4) + 4 - 1 - 4 = 0$

$(x-1)^2 + (y-2)^2 = 1$

Center $(1,2)$, radius $1$

Step 2: Tangent to parabola

For $y^2 = 4x$, tangent is $y = mx + \frac{1}{m}$

This line should also be tangent to the circle.

Step 3: Distance from circle center to line

Line: $y = mx + \frac{1}{m} \Rightarrow mx - y + \frac{1}{m} = 0$

Distance from $(1,2)$ to line = $\frac{|m(1) - 2 + \frac{1}{m}|}{\sqrt{m^2 + 1}} = 1$ (radius)

Step 4: Solve for m

$\frac{|m - 2 + \frac{1}{m}|}{\sqrt{m^2 + 1}} = 1$

Square both sides and solve the equation to find possible slopes.

ðŸŽ¯ Strategy: Circle-Parabola Common Tangents

  • Use standard tangent form for parabola: $y = mx + \frac{a}{m}$ for $y^2 = 4ax$
  • Apply distance from circle center to this line = radius
  • Solve the resulting equation in $m$
  • Check if solutions give real tangents (discriminant conditions)

Type 3: Common Tangents to Two Parabolas

Find the common tangent to the parabolas $y^2 = 4ax$ and $x^2 = 4ay$

Solution Approach:

Step 1: Tangents in slope form

For $y^2 = 4ax$: tangent is $y = mx + \frac{a}{m}$

For $x^2 = 4ay$: tangent is $y = mx - am^2$ (since it's of form $y = mx + c$ with $c = -am^2$)

Step 2: Equate intercepts for common tangent

For the line to be tangent to both, we need:

$\frac{a}{m} = -am^2$

Assuming $a \neq 0$, divide by $a$: $\frac{1}{m} = -m^2$

Step 3: Solve for m

$\frac{1}{m} = -m^2 \Rightarrow 1 = -m^3 \Rightarrow m^3 = -1 \Rightarrow m = -1$

Step 4: Find the tangent equation

Using $m = -1$ in either form:

$y = (-1)x + \frac{a}{-1} = -x - a$

Or $y = (-1)x - a(-1)^2 = -x - a$

So common tangent is $x + y + a = 0$

ðŸ“ˆ Geometric Insight

The parabolas $y^2 = 4ax$ and $x^2 = 4ay$ are reflections of each other about the line $y = x$. Their common tangent is symmetric with respect to this line.

ðŸ”„ Systematic Problem Solving Framework

Step 1: Curve Analysis

  • Identify curve types and their standard forms
  • Note centers, foci, vertices, radii as applicable
  • Determine relative positions of curves

Step 2: Tangent Form Selection

  • Choose appropriate tangent form (slope, point, parametric)
  • Consider symmetry to reduce calculations
  • Use the simpler curve's tangent form first

Step 3: Condition Application

  • Apply tangency condition to second curve
  • Use distance formulas for circles
  • Use discriminant = 0 for conics

Step 4: Solution & Verification

  • Solve the resulting equation system
  • Check if solutions satisfy all conditions
  • Verify geometric interpretation makes sense
