Problem 1: Leibniz Rule Application

Evaluate $I(\alpha) = \int_0^{\pi} \frac{dx}{1 + \alpha \cos x}$ for $|\alpha| < 1$

Solution Using Leibniz Rule:

Step 1: Differentiate with respect to $\alpha$:

$$I'(\alpha) = -\int_0^{\pi} \frac{\cos x}{(1 + \alpha \cos x)^2} dx$$

Step 2: Use substitution $t = \tan\frac{x}{2}$:

$$I(\alpha) = \frac{\pi}{\sqrt{1-\alpha^2}}$$

Step 3: Verify boundary condition $I(0) = \pi$

Final Answer: $I(\alpha) = \frac{\pi}{\sqrt{1-\alpha^2}}$

JEE Advanced 2022 Hard

Problem 2: Feynman's Technique

Evaluate $I(a) = \int_0^{\infty} \frac{\tan^{-1}(ax)}{x(1+x^2)} dx$

Feynman's Method:

Step 1: Differentiate with respect to $a$:

$$I'(a) = \int_0^{\infty} \frac{dx}{(1+x^2)(1+a^2x^2)}$$

Step 2: Partial fractions:

$$\frac{1}{(1+x^2)(1+a^2x^2)} = \frac{1}{1-a^2}\left(\frac{1}{1+x^2} - \frac{a^2}{1+a^2x^2}\right)$$

Step 3: Integrate:

$$I'(a) = \frac{\pi}{2(1+a)}$$

Step 4: Integrate with $I(0) = 0$:

$$I(a) = \frac{\pi}{2} \ln(1+a)$$

JEE Advanced 2021 Medium

Problem 3: Symmetry Method

Evaluate $I = \int_0^{\pi} \frac{x \sin x}{1 + \cos^2 x} dx$

Using King's Property:

Step 1: Use property $\int_a^b f(x)dx = \int_a^b f(a+b-x)dx$

Step 2: Apply to our integral:

$$I = \int_0^{\pi} \frac{(\pi - x) \sin x}{1 + \cos^2 x} dx$$

Step 3: Add original and transformed integral:

$$2I = \pi \int_0^{\pi} \frac{\sin x}{1 + \cos^2 x} dx$$

Step 4: Substitute $t = \cos x$:

$$I = \frac{\pi}{2} \int_{-1}^1 \frac{dt}{1+t^2} = \frac{\pi}{2} [\tan^{-1} t]_{-1}^1 = \frac{\pi^2}{4}$$

When to Use Which Method:

Leibniz Rule

Integrals with parameter in limits or integrand that simplify on differentiation

Feynman's Technique

Tough integrals where introducing a parameter creates solvable DE

Symmetry Methods

Definite integrals over symmetric intervals $[-a,a]$ or $[0,a]$

Common Patterns to Recognize:

  • $\int_0^{\pi} x f(\sin x) dx = \frac{\pi}{2} \int_0^{\pi} f(\sin x) dx$
  • $\int_a^b f(x) dx = \int_a^b f(a+b-x) dx$
  • $\int_0^{\pi/2} f(\sin x) dx = \int_0^{\pi/2} f(\cos x) dx$
  • $\int_0^{\infty} \frac{f(ax) - f(bx)}{x} dx = [f(0) - f(\infty)] \ln\frac{b}{a}$

üìö Leibniz Rule Formula Sheet

Standard Form:

$$\frac{d}{d\alpha}\int_{a(\alpha)}^{b(\alpha)} f(x,\alpha)dx = f(b(\alpha),\alpha)b'(\alpha) - f(a(\alpha),\alpha)a'(\alpha) + \int_{a(\alpha)}^{b(\alpha)} \frac{\partial f}{\partial \alpha} dx$$

Special Cases:

  • Constant limits: First two terms vanish
  • $f(x,\alpha) = g(x)h(\alpha)$: Product rule applies
  • Symmetric limits: Often simplifies dramatically
