Problem 1: Basic Piecewise Function

$$f(x) = \begin{cases} x^2 & \text{if } x < 0 \\ x + 1 & \text{if } 0 \leq x \leq 2 \\ 5 - x & \text{if } x > 2 \end{cases}$$

Find the domain and range of $f(x)$

Solution Approach:

Step 1: Domain Analysis

‚Ä¢ All pieces are defined for all real numbers in their intervals

‚Ä¢ Domain: $(-\infty, \infty)$

Step 2: Range of Each Piece

‚Ä¢ For $x < 0$: $f(x) = x^2$, range: $(0, \infty)$

‚Ä¢ For $0 \leq x \leq 2$: $f(x) = x + 1$, range: $[1, 3]$

‚Ä¢ For $x > 2$: $f(x) = 5 - x$, range: $(-\infty, 3)$

Step 3: Combine Ranges

‚Ä¢ Union: $(0, \infty) \cup [1, 3] \cup (-\infty, 3) = (-\infty, \infty)$

‚Ä¢ But check boundaries: At $x=0$, $f(0)=1$ from second piece

‚Ä¢ Range: $(-\infty, \infty)$

Problem 2: Absolute Value as Piecewise

$$f(x) = |x - 1| + |x - 2|$$

Find the domain and range of $f(x)$

Solution Approach:

Step 1: Convert to Piecewise

Critical points at $x=1$ and $x=2$

$$f(x) = \begin{cases} -(x-1)-(x-2) = -2x+3 & \text{if } x < 1 \\ (x-1)-(x-2) = 1 & \text{if } 1 \leq x < 2 \\ (x-1)+(x-2) = 2x-3 & \text{if } x \geq 2 \end{cases}$$

Step 2: Domain and Range Analysis

‚Ä¢ Domain: $(-\infty, \infty)$

‚Ä¢ For $x < 1$: $f(x) = -2x+3$, range: $(1, \infty)$

‚Ä¢ For $1 \leq x < 2$: $f(x) = 1$, range: $\{1\}$

‚Ä¢ For $x \geq 2$: $f(x) = 2x-3$, range: $[1, \infty)$

Step 3: Combine Results

‚Ä¢ Union: $(1, \infty) \cup \{1\} \cup [1, \infty) = [1, \infty)$

‚Ä¢ Range: $[1, \infty)$

Problem 3: Greatest Integer Function

$$f(x) = [x] + \sqrt{x - [x]}$$

where $[x]$ denotes the greatest integer less than or equal to $x$

Find the domain and range of $f(x)$

Solution Approach:

Step 1: Understand the Function

Let $x = n + r$ where $n = [x]$ and $0 \leq r < 1$

Then $f(x) = n + \sqrt{r}$

Step 2: Domain Analysis

‚Ä¢ $\sqrt{x - [x]}$ requires $x - [x] \geq 0$ which is always true

‚Ä¢ Domain: $(-\infty, \infty)$

Step 3: Range Analysis

For each integer $n$, when $x \in [n, n+1)$:

‚Ä¢ $f(x) = n + \sqrt{r}$ where $r \in [0, 1)$

‚Ä¢ Range for this interval: $[n, n+1)$

‚Ä¢ Union over all integers: $\bigcup_{n=-\infty}^{\infty} [n, n+1) = (-\infty, \infty)$

Step 4: Final Answer

‚Ä¢ Domain: $(-\infty, \infty)$

‚Ä¢ Range: $(-\infty, \infty)$

üöÄ Advanced Piecewise Strategies

Common Piecewise Patterns:

  • Absolute Value: $|x-a|$ creates piecewise at $x=a$
  • Signum Function: $\text{sgn}(x)$ has 3 pieces
  • Greatest Integer: $[x]$ changes at every integer
  • Fractional Part: $\{x\} = x - [x]$

Boundary Checks:

  • Always check function values at interval endpoints
  • Verify if boundary values are included/excluded
  • Check continuity/discontinuity at boundaries
  • Look for gaps in the range
