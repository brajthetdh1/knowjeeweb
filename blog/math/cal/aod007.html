1. Absolute Error

Fundamental Basic

What is Absolute Error?

Absolute error is the magnitude of the difference between the measured value and the true value of a quantity.

Formula

$$ \Delta x = |x_{\text{measured}} - x_{\text{true}}| $$

Where:

  • $\Delta x$ = Absolute error
  • $x_{\text{measured}}$ = Measured value
  • $x_{\text{true}}$ = True value (or accepted value)

üìù Example 1: Simple Absolute Error

Problem: The true length of a rod is 15.2 cm. A student measures it as 15.5 cm. Calculate the absolute error.

Step 1: Identify values:

True value = 15.2 cm

Measured value = 15.5 cm

Step 2: Apply formula:

$\Delta x = |15.5 - 15.2| = |0.3| = 0.3$ cm

Absolute Error = 0.3 cm

üìù Example 2: Multiple Measurements

Problem: A student measures the length of a table 3 times: 152 cm, 151 cm, and 153 cm. The true length is 150 cm. Find mean absolute error.

Step 1: Calculate individual absolute errors:

$|\Delta x_1| = |152 - 150| = 2$ cm

$|\Delta x_2| = |151 - 150| = 1$ cm

$|\Delta x_3| = |153 - 150| = 3$ cm

Step 2: Mean absolute error:

$\Delta x_{\text{mean}} = \frac{2 + 1 + 3}{3} = 2$ cm

Mean Absolute Error = 2 cm

üí° Key Points

  • Absolute error has the same units as the measured quantity
  • It tells you how far off your measurement is
  • Smaller absolute error = more accurate measurement
  • For multiple measurements, use mean absolute error

2. Relative Error

Important Intermediate

What is Relative Error?

Relative error is the ratio of absolute error to the true value. It gives the proportion of error relative to the size of measurement.

Formula

$$ \text{Relative Error} = \frac{\Delta x}{x_{\text{true}}} $$

Where:

  • $\Delta x$ = Absolute error
  • $x_{\text{true}}$ = True value

üìù Example 3: Relative Error Calculation

Problem: A student measures a 200 cm rod as 198 cm. Calculate the relative error.

Step 1: Calculate absolute error:

$\Delta x = |198 - 200| = 2$ cm

Step 2: Apply relative error formula:

Relative Error = $\frac{2}{200} = 0.01$

Relative Error = 0.01

üìù Example 4: Comparing Measurements

Problem: Which measurement is more precise?

Measurement A: 10 ¬± 0.1 cm

Measurement B: 100 ¬± 0.5 cm

Step 1: Calculate relative errors:

For A: $\frac{0.1}{10} = 0.01$

For B: $\frac{0.5}{100} = 0.005$

Step 2: Compare:

Measurement B has smaller relative error (0.005 < 0.01)

Measurement B is more precise

üí° Key Points

  • Relative error is a dimensionless quantity (no units)
  • It allows comparison of precision across different measurements
  • Smaller relative error = more precise measurement
  • Useful when true values have different magnitudes

3. Percentage Error

Fundamental Basic

What is Percentage Error?

Percentage error is the relative error expressed as a percentage. It's the most commonly used error measure in practical applications.

Formula

$$ \text{Percentage Error} = \left(\frac{\Delta x}{x_{\text{true}}}\right) \times 100\% $$

Or equivalently:

$$ \text{Percentage Error} = \text{Relative Error} \times 100\% $$

üìù Example 5: Percentage Error Calculation

Problem: The accepted value of gravitational acceleration is 9.8 m/s¬≤. A student calculates it as 9.5 m/s¬≤. Find percentage error.

Step 1: Calculate absolute error:

$\Delta x = |9.5 - 9.8| = 0.3$ m/s¬≤

Step 2: Calculate percentage error:

Percentage Error = $\frac{0.3}{9.8} \times 100\% \approx 3.06\%$

Percentage Error ‚âà 3.06%

üìù Example 6: Practical Application

Problem: A 500 ml flask is measured as 495 ml. Express the error as percentage error.

Step 1: Absolute error:

$\Delta x = |495 - 500| = 5$ ml

Step 2: Percentage error:

Percentage Error = $\frac{5}{500} \times 100\% = 1\%$

The measurement has 1% error

üí° Key Points

  • Percentage error is dimensionless (expressed as %)
  • Most intuitive way to express measurement accuracy
  • Industry standard: measurements often specified as "value ¬± % error"
  • Easy to compare across different types of measurements

Essential Error Formulas

5. Advanced JEE Examples

JEE Advanced Advanced

üìù Example 7: JEE Main 2022

Problem: The period of oscillation of a simple pendulum is given by $T = 2\pi\sqrt{\frac{L}{g}}$. If the measurements of L and g have errors of 2% and 1% respectively, what is the percentage error in T?

Step 1: Write the formula:

$T = 2\pi\sqrt{\frac{L}{g}} = 2\pi L^{1/2} g^{-1/2}$

Step 2: Use error propagation formula:

$\frac{\Delta T}{T} = \frac{1}{2}\frac{\Delta L}{L} + \frac{1}{2}\frac{\Delta g}{g}$

Step 3: Substitute percentage errors:

$\frac{\Delta T}{T} \times 100\% = \frac{1}{2}(2\%) + \frac{1}{2}(1\%) = 1\% + 0.5\% = 1.5\%$

Percentage error in T = 1.5%

üìù Example 8: JEE Main 2021

Problem: The density of a cube is measured using the formula $\rho = \frac{m}{L^3}$. If percentage errors in m and L are 1% and 2% respectively, find percentage error in œÅ.

Step 1: Write the formula:

$\rho = m L^{-3}$

Step 2: Use error propagation:

$\frac{\Delta \rho}{\rho} = \frac{\Delta m}{m} + 3\frac{\Delta L}{L}$

Step 3: Convert to percentages:

$\frac{\Delta \rho}{\rho} \times 100\% = 1\% + 3(2\%) = 1\% + 6\% = 7\%$

Percentage error in œÅ = 7%

Problem 1: A student measures a 50 cm rod as 49.5 cm. Calculate:

a) Absolute error

b) Relative error

c) Percentage error

Problem 2: Which measurement is more precise?

Measurement X: 25 ¬± 0.5 kg

Measurement Y: 100 ¬± 1.0 kg

Problem 3: The volume of a sphere is $V = \frac{4}{3}\pi R^3$. If radius is measured with 3% error, what is the percentage error in volume?

Problem 4: A student measures current (I) and resistance (R) to calculate power using $P = I^2R$. If errors in I and R are 2% and 1% respectively, find error in P.

Answers

Problem 1:

a) 0.5 cm

b) 0.01

c) 1%

Problem 3: 9%

Problem 4: 5%

7. JEE Preparation Tips

üöÄ Exam Strategy

Common Mistakes to Avoid

  • Forgetting to take absolute value in error calculations
  • Confusing precision with accuracy
  • Mishandling error propagation in formulas
  • Not using percentage error for comparisons
  • Ignoring significant figures in final answers

Quick Solving Techniques

  • For $y = x^n$: $\frac{\Delta y}{y} = n\frac{\Delta x}{x}$
  • For $y = ab$: $\frac{\Delta y}{y} = \frac{\Delta a}{a} + \frac{\Delta b}{b}$
  • For $y = \frac{a}{b}$: $\frac{\Delta y}{y} = \frac{\Delta a}{a} + \frac{\Delta b}{b}$
  • Always express final error with 1-2 significant figures
