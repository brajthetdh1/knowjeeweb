Random Variables: Discrete vs Continuous

A random variable is a variable whose possible values are numerical outcomes of a random phenomenon.

Discrete Random Variables

  • Countable number of values
  • Probability Mass Function (PMF)
  • Examples: Number of heads in coin tosses

Continuous Random Variables

  • Uncountable values in an interval
  • Probability Density Function (PDF)
  • Examples: Height, weight, time

Key Formulas:

Expectation: $E(X) = \sum x_i P(X=x_i)$ (Discrete)

$E(X) = \int_{-\infty}^{\infty} x f(x) dx$ (Continuous)

Variance: $Var(X) = E(X^2) - [E(X)]^2$

Binomial Distribution

Models the number of successes in a fixed number of independent Bernoulli trials.

Binomial Distribution Formula:

$P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}$

Where:

  • $n$ = number of trials
  • $k$ = number of successes
  • $p$ = probability of success
  • $\binom{n}{k} = \frac{n!}{k!(n-k)!}$

Example Problem:

A fair coin is tossed 10 times. Find the probability of getting exactly 6 heads.

Solution:

$n = 10$, $k = 6$, $p = 0.5$

$P(X=6) = \binom{10}{6} (0.5)^6 (0.5)^4 = 210 \times 0.015625 \times 0.0625 = 0.205$

Normal Distribution

The most important continuous probability distribution, characterized by bell-shaped curve.

Normal Distribution Properties:

PDF: $f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2}$

Where $\mu$ = mean, $\sigma$ = standard deviation

  • Bell-shaped and symmetric about mean
  • 68-95-99.7 Rule:
    • 68% within $\mu \pm \sigma$
    • 95% within $\mu \pm 2\sigma$
    • 99.7% within $\mu \pm 3\sigma$

Standard Normal Distribution:

$Z = \frac{X - \mu}{\sigma}$ transforms any normal distribution to standard normal with $\mu=0$, $\sigma=1$
