JEE ADVANCED PROBLEM

Newton's Laws of Motion

PROBLEM STATEMENT

A block of mass m = 5 kg is placed on a rough inclined plane with angle θ = 37°. The coefficients of friction are μₛ = 0.5 and μₖ = 0.3. Determine if the block will slide down, and if yes, find its acceleration.

Given Data

• Mass: m = 5 kg
• Angle: θ = 37°
• Static friction: μₛ = 0.5
• Kinetic friction: μₖ = 0.3
• g = 10 m/s²

Required

• Will block slide? ✓
• Acceleration if slides ✓

STEP 1: CHECK IF BLOCK WILL SLIDE

Maximum Static Friction

fs,max = μsN
N = mg cosθ
fs,max = μsmg cosθ

Component Down the Plane

Fdown = mg sinθ
Compare: mg sinθ vs μsmg cosθ

Calculation

sin37° = 0.6, cos37° = 0.8
mg sinθ = 5×10×0.6 = 30 N
fs,max = 0.5×5×10×0.8 = 20 N
30 N > 20 N
∴ BLOCK WILL SLIDE!

STEP 2: FREE BODY DIAGRAM

Forces Acting

• Weight: mg ↓
• Normal: N ⟂
• Kinetic friction: fk ↗
• Component: mg sinθ ↘
• Component: mg cosθ ↙

Along the Incline

ma = mg sinθ - fk

STEP 3: APPLY NEWTON'S SECOND LAW

Along Incline (x-direction)

ΣFx = ma
mg sinθ - fk = ma
mg sinθ - μkN = ma

Perpendicular to Incline (y-direction)

ΣFy = 0
N - mg cosθ = 0
N = mg cosθ

Combine Equations

Substitute N = mg cosθ into x-equation:
mg sinθ - μkmg cosθ = ma
a = g(sinθ - μkcosθ)

STEP 4: FINAL CALCULATION

Substitute Values

a = g(sinθ - μkcosθ)
a = 10(0.6 - 0.3×0.8)
a = 10(0.6 - 0.24)
a = 10 × 0.36
a = 3.6 m/s²

✓ Resolve forces properly
