Question 1

Easy

Prove the identity:

$\sin^2\theta + \cos^2\theta = 1$

Solution:

Step 1: Consider a right triangle

In a right triangle with hypotenuse $h$, opposite side $o$, and adjacent side $a$:

$\sin\theta = \frac{o}{h}$, $\cos\theta = \frac{a}{h}$

Step 2: Square both sides

$\sin^2\theta = \frac{o^2}{h^2}$

$\cos^2\theta = \frac{a^2}{h^2}$

Step 3: Add the equations

$\sin^2\theta + \cos^2\theta = \frac{o^2}{h^2} + \frac{a^2}{h^2} = \frac{o^2 + a^2}{h^2}$

Step 4: Apply Pythagorean theorem

By Pythagorean theorem: $o^2 + a^2 = h^2$

Therefore: $\sin^2\theta + \cos^2\theta = \frac{h^2}{h^2} = 1$

Hence Proved: $\sin^2\theta + \cos^2\theta = 1$

Question 2

Medium

Find the general solution of:

$\sin 2\theta = \frac{\sqrt{3}}{2}$

Solution:

Step 1: Identify principal values

$\sin\alpha = \frac{\sqrt{3}}{2}$

Principal values: $\alpha = 60° = \frac{\pi}{3}$ or $\alpha = 120° = \frac{2\pi}{3}$

Step 2: Write general solution for sin

If $\sin x = \sin\alpha$, then $x = n\pi + (-1)^n\alpha$

Here, $2\theta = n\pi + (-1)^n \cdot \frac{\pi}{3}$

Step 3: Solve for θ

$\theta = \frac{n\pi}{2} + (-1)^n \cdot \frac{\pi}{6}$

Step 4: Alternative form

When $n$ is even: $\theta = \frac{n\pi}{2} + \frac{\pi}{6}$

When $n$ is odd: $\theta = \frac{n\pi}{2} - \frac{\pi}{6}$

Answer: $\theta = \frac{n\pi}{2} + (-1)^n \cdot \frac{\pi}{6}$, where $n \in \mathbb{Z}$

Question 3

Easy

If $\sin A = \frac{3}{5}$ and $A$ is in the first quadrant, find $\cos A$ and $\tan A$

Solution:

Step 1: Use fundamental identity

$\sin^2 A + \cos^2 A = 1$

$\left(\frac{3}{5}\right)^2 + \cos^2 A = 1$

Step 2: Solve for cos A

$\frac{9}{25} + \cos^2 A = 1$

$\cos^2 A = 1 - \frac{9}{25} = \frac{16}{25}$

$\cos A = \pm\frac{4}{5}$

Step 3: Apply quadrant condition

Since $A$ is in first quadrant, both sin and cos are positive

$\cos A = +\frac{4}{5}$

Step 4: Find tan A

$\tan A = \frac{\sin A}{\cos A} = \frac{3/5}{4/5} = \frac{3}{4}$

Answer: $\cos A = \frac{4}{5}$, $\tan A = \frac{3}{4}$

Question 4

Hard

Prove that:

$\frac{\sin A + \sin B}{\cos A + \cos B} = \tan\left(\frac{A+B}{2}\right)$

Solution:

Step 1: Apply sum-to-product formulas

$\sin A + \sin B = 2\sin\left(\frac{A+B}{2}\right)\cos\left(\frac{A-B}{2}\right)$

$\cos A + \cos B = 2\cos\left(\frac{A+B}{2}\right)\cos\left(\frac{A-B}{2}\right)$

Step 2: Substitute in LHS

$\frac{\sin A + \sin B}{\cos A + \cos B} = \frac{2\sin\left(\frac{A+B}{2}\right)\cos\left(\frac{A-B}{2}\right)}{2\cos\left(\frac{A+B}{2}\right)\cos\left(\frac{A-B}{2}\right)}$

Step 3: Cancel common terms

$= \frac{\sin\left(\frac{A+B}{2}\right)}{\cos\left(\frac{A+B}{2}\right)}$

Step 4: Simplify

$= \tan\left(\frac{A+B}{2}\right)$ = RHS

Hence Proved

Question 5

Medium

Find the value of:

$\tan^{-1}(1) + \cos^{-1}\left(-\frac{1}{2}\right)$

Solution:

Step 1: Find tan⁻¹(1)

We need angle whose tangent is 1

$\tan\theta = 1 \Rightarrow \theta = 45° = \frac{\pi}{4}$

$\tan^{-1}(1) = \frac{\pi}{4}$

Step 2: Find cos⁻¹(-1/2)

We need angle whose cosine is $-\frac{1}{2}$

$\cos\theta = -\frac{1}{2}$

In range $[0, \pi]$: $\theta = 120° = \frac{2\pi}{3}$

$\cos^{-1}\left(-\frac{1}{2}\right) = \frac{2\pi}{3}$

Step 3: Add the values

$\tan^{-1}(1) + \cos^{-1}\left(-\frac{1}{2}\right) = \frac{\pi}{4} + \frac{2\pi}{3}$

$= \frac{3\pi}{12} + \frac{8\pi}{12} = \frac{11\pi}{12}$

Answer: $\frac{11\pi}{12}$ or $165°$

Question 6

Hard

Simplify:

$\sin(x+y)\sin(x-y)$

Solution:

Step 1: Expand using compound angle formulas

$\sin(x+y) = \sin x\cos y + \cos x\sin y$

$\sin(x-y) = \sin x\cos y - \cos x\sin y$

Step 2: Multiply using (a+b)(a-b) = a² - b²

$\sin(x+y)\sin(x-y) = (\sin x\cos y)^2 - (\cos x\sin y)^2$

$= \sin^2 x\cos^2 y - \cos^2 x\sin^2 y$

Step 3: Factor out common terms

$= \sin^2 x\cos^2 y - \cos^2 x\sin^2 y$

Step 4: Alternative form using identities

$= \sin^2 x(1-\sin^2 y) - (1-\sin^2 x)\sin^2 y$

$= \sin^2 x - \sin^2 x\sin^2 y - \sin^2 y + \sin^2 x\sin^2 y$

$= \sin^2 x - \sin^2 y$

Answer: $\sin^2 x - \sin^2 y$ or $\sin^2 x\cos^2 y - \cos^2 x\sin^2 y$

Question 7

Medium

Express $\sin 3\theta$ in terms of $\sin\theta$

Solution:

Step 1: Write as sum

$\sin 3\theta = \sin(2\theta + \theta)$

Step 2: Apply sum formula

$= \sin 2\theta\cos\theta + \cos 2\theta\sin\theta$

Step 3: Use double angle formulas

$\sin 2\theta = 2\sin\theta\cos\theta$

$\cos 2\theta = 1 - 2\sin^2\theta$

Step 4: Substitute

$= (2\sin\theta\cos\theta)\cos\theta + (1-2\sin^2\theta)\sin\theta$

$= 2\sin\theta\cos^2\theta + \sin\theta - 2\sin^3\theta$

$= 2\sin\theta(1-\sin^2\theta) + \sin\theta - 2\sin^3\theta$

$= 2\sin\theta - 2\sin^3\theta + \sin\theta - 2\sin^3\theta$

$= 3\sin\theta - 4\sin^3\theta$

Answer: $\sin 3\theta = 3\sin\theta - 4\sin^3\theta$

Question 8

Easy

Find the value of:

$\sin 15° \times \cos 15°$

Solution:

Step 1: Use double angle formula

We know: $\sin 2\theta = 2\sin\theta\cos\theta$

Therefore: $\sin\theta\cos\theta = \frac{\sin 2\theta}{2}$

Step 2: Apply with θ = 15°

$\sin 15°\cos 15° = \frac{\sin(2 \times 15°)}{2}$

$= \frac{\sin 30°}{2}$

Step 3: Evaluate

$= \frac{1/2}{2} = \frac{1}{4}$

Answer: $\frac{1}{4}$
