The Limit Concept

A limit describes the value that a function approaches as the input approaches some value. Limits are essential for defining derivatives and integrals.

\[ \lim_{x \to 1} f(x) = 3 \]

Adjust the parameters to see how the limit changes. Notice how the function approaches the limit value.

For a limit to exist, the left-hand limit and right-hand limit must be equal.

Understanding One-Sided Limits

One-sided limits examine the behavior of a function as it approaches a point from only one direction - either from the left or from the right.

Left limit: 1, Right limit: 2

Limit does not exist at x = 0

For a limit to exist at a point, the left-hand and right-hand limits must be equal.

Discontinuities often occur when one-sided limits are different.

Practice Problem

JEE Advanced Practice Question

Let \( f(x) = \begin{cases} x^2 + 1 & \text{if } x < 1 \\ ax + b & \text{if } x \geq 1 \end{cases} \)
If f(x) is continuous at x = 1, find the values of a and b.

Solution:

For continuity at x = 1:
1. Left-hand limit: \( \lim_{x \to 1^-} f(x) = 1^2 + 1 = 2 \)
2. Right-hand limit: \( \lim_{x \to 1^+} f(x) = a(1) + b = a + b \)
3. f(1) = a + b
For continuity: 2 = a + b
Also, the function must be differentiable at x = 1 for a unique solution.

Answer: a = 2, b = 0
