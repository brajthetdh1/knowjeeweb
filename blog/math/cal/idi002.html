JEE Main tests method application while JEE Advanced tests pattern recognition + creative thinking

JEE Main 2023 Mains Pattern

Pattern 1: Basic Substitution with Algebraic Manipulation

Evaluate $\int \frac{x^3+1}{x+1} dx$

Mains Approach (2-3 steps):

Step 1: Factor numerator: $x^3+1 = (x+1)(x^2-x+1)$

Step 2: Simplify: $\int (x^2-x+1) dx$

Step 3: Integrate: $\frac{x^3}{3} - \frac{x^2}{2} + x + C$

Advanced Variation:

$\int \frac{x^5+1}{x^2+1} dx$ (Requires polynomial division first)

Tests both algebraic manipulation and integration skills

JEE Advanced 2022 Advanced Pattern

Pattern 2: Trigonometric Substitution with Multiple Steps

Evaluate $\int \frac{dx}{x\sqrt{x^2+x+1}}$

Advanced Approach (4-6 steps):

Step 1: Complete square: $x^2+x+1 = (x+\frac{1}{2})^2 + \frac{3}{4}$

Step 2: Substitute $x+\frac{1}{2} = \frac{\sqrt{3}}{2}\tan\theta$

Step 3: Transform integral to trigonometric form

Step 4: Use trigonometric identities

Step 5: Integrate and back-substitute

Mains Equivalent:

$\int \frac{dx}{x\sqrt{x^2+4}}$ (Direct substitution $x=2\tan\theta$)

Simpler substitution with fewer steps

JEE Main 2024 Mains Pattern

Pattern 3: Partial Fractions with Repeated Roots

Evaluate $\int \frac{x^2+1}{(x-1)^2(x+2)} dx$

Mains Approach:

Step 1: Set up partial fractions: $\frac{A}{x-1} + \frac{B}{(x-1)^2} + \frac{C}{x+2}$

Step 2: Find A, B, C by comparing coefficients

Step 3: Integrate each term separately
