Simple Pendulum

A simple pendulum consists of a point mass suspended by a massless, inextensible string.

Time Period Formula

T = 2π√(L/g)

For small oscillations (θ < 10°)

Key Features:

• Independent of mass

• Depends only on L and g

• SHM for small angles

Components:

Bob

Length L

Oscillation

Compound Pendulum

Physical Pendulum

✓ Rigid body oscillating

✓ Axis not through COM

✓ T = 2π√(I/mgh)

✓ I = Moment of Inertia

✓ h = distance from pivot to COM

Equivalent Length

✓ Leq = I/mh

✓ Same period as simple pendulum

✓ Center of oscillation concept

✓ Important for JEE Advanced

JEE Memory Tip: For uniform rod pivoted at end, T = 2π√(2L/3g) and Leq = 2L/3

Conical Pendulum

Time Period Formula

T = 2π√(Lcosθ/g)

Independent of mass, depends on θ

Key Relationships

✓ Tension T = mg/cosθ

✓ Angular speed ω = √(g/Lcosθ)

✓ Radius r = Lsinθ

✓ Speed v = √(rg tanθ)

Characteristics

✓ Bob moves in horizontal circle

✓ String traces conical surface

✓ Uniform circular motion

✓ Constant time period

Spring-Mass System

Time Period Formula

T = 2π√(m/k)

Depends on mass and spring constant

✓ Series: 1/keq = 1/k₁ + 1/k₂

✓ Parallel: keq = k₁ + k₂

✓ Series: T = 2π√(m/keq)

✓ Energy: E = ½kA²

✓ PEmax = ½kA²

JEE Example Problem 1

JEE Main 2022 Question:

A simple pendulum has time period T. What happens to T when the pendulum is taken to a height equal to Earth's radius above the surface?

Solution:

Step 1: Gravity variation with height

g' = g/(1 + h/R)²

At h = R: g' = g/4

Step 2: Time period formula

T = 2π√(L/g)

T' = 2π√(L/g')

Step 3: Ratio calculation

T'/T = √(g/g') = √4 = 2

Time period doubles

JEE Example Problem 2

JEE Advanced 2020 Question:

A spring-mass system has time period T. The spring is cut into two equal parts and connected in parallel. Find the new time period with the same mass.

Solution:

Step 1: Spring constant for half spring

For spring: k ∝ 1/L

k' = 2k (when length halved)

Step 2: Parallel combination

keq = k' + k' = 4k

Step 3: New time period

T = 2π√(m/k)

T' = 2π√(m/4k) = T/2

Time period halves
