What is Acceleration?

Acceleration is the rate of change of velocity with time.

It's a vector quantity with both magnitude and direction.

Key Formula:

a = Δv/Δt

SI Unit: m/s²

JEE Significance: Central to all kinematics and dynamics problems.

Key aspects:

Rate

Vector

Change

Types of Acceleration

Uniform Acceleration

✓ Constant rate

✓ Straight-line motion

✓ Example: Free fall

✓ Equations of motion apply

Non-uniform Acceleration

✓ Changing rate

✓ Calculus required

✓ Example: Spring motion

✓ JEE Advanced favorite

JEE Memory Tip: Uniform acceleration = constant slope on v-t graph

Acceleration Components

Tangential Acceleration

✓ Changes speed

✓ Along velocity direction

✓ aₜ = dv/dt

✓ Linear acceleration

Centripetal Acceleration

✓ Changes direction

✓ Towards center

✓ a꜀ = v²/r

✓ Circular motion

Total Acceleration

a = √(aₜ² + a꜀²)

Vector sum of tangential and centripetal components

Equations of Motion

For uniform acceleration in straight line:

v = u + at

Final velocity

s = ut + ½at²

Displacement

v² = u² + 2as

Velocity-position

s = ½(u+v)t

Average velocity

Remember: These equations work ONLY for constant acceleration!

JEE Example Problem 1

JEE Main 2023 Question:

A car accelerates uniformly from rest to 20 m/s in 8 seconds. Find acceleration and distance covered.

Solution:

Step 1: Acceleration

u = 0, v = 20 m/s, t = 8s

a = (v-u)/t = (20-0)/8 = 2.5 m/s²

Step 2: Distance

s = ut + ½at² = 0 + ½×2.5×8²

s = ½×2.5×64 = 80 meters

JEE Example Problem 2

JEE Advanced 2021 Question:

A particle moves with acceleration a = 3t². If it starts from rest, find velocity at t=2s and displacement at t=2s.

Solution:

Step 1: Velocity

a = dv/dt = 3t²

∫dv = ∫3t² dt

v = t³ + C

At t=0, v=0 ⇒ C=0

v(2) = 2³ = 8 m/s

Step 2: Displacement

v = dx/dt = t³

∫dx = ∫t³ dt

x = t⁴/4 + C

At t=0, x=0 ⇒ C=0

x(2) = 16/4 = 4 meters

Graphical Analysis

v-t Graph

✓ Slope = Acceleration

✓ Area = Displacement

✓ Straight line = Constant acceleration

a-t Graph

✓ Area = Change in velocity

✓ Constant line = Uniform acceleration

✓ Curve = Variable acceleration

JEE Strategy: Master graph interpretation - it's frequently tested in both JEE Main and Advanced!

Relative Acceleration

Relative Acceleration Formula

a₁₂ = a₁ - a₂

Acceleration of 1 relative to 2

Same Direction

✓ a₁₂ = |a₁ - a₂|

✓ Magnitude difference

Opposite Direction

✓ a₁₂ = a₁ + a₂

✓ Magnitude sum

Free Fall & Gravity

Acceleration due to Gravity

g = 9.8 m/s²

✓ Always downward

✓ Independent of mass

✓ Constant near Earth's surface

Upward Motion

✓ a = -g

✓ Velocity decreases

✓ Maximum height at v=0

Downward Motion

✓ a = +g

✓ Velocity increases

✓ Symmetric motion
